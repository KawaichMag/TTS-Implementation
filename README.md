# TTS-Implementation
Project for study purposes

Current working TTS version is allowed in `src/GAI_model.ipynb`.
To launch model training you should specify list of audios from LJSpeech-1.1 dataset with transcription presented in original `metadata.csv` file.
For such purposes we consider to use `uv`, all dependancies are in `pyproject.toml` file.

```bash
uv sync
```
